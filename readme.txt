== Installation ==

1. Upload directory to server
2. If your server does not handle permissions automatically then chmod 777 the data.txt file (if you change the data file then chmod 777 that file)
3. Configure the square variables as in the demo.
4. Default password is password. you can change that with the passwords variable as in the example file.
